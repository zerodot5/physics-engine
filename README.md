# physics-engine
a barebones 2d physics engine for my cs class final.
now completed with all the features I wanted it to have.

# binds
Space - spawn planet (prompts for velocity and mass)
Enter - spawn fixed star (prompts for mass)
P - pause/resume
Tab - toggle planet focus mode
Q/E - cycle focused planet (focus mode only)
Arrow keys - pan camera
Z/X - zoom in/out
G - change gravity coefficient
T - change time step
C - clear all planets
Escape - close
